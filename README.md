# hq-mksports-resource

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态资源的形式存储，不针对任何特定网站或域名，仅作为页面文件的集中存放与版本管理用途。

## 目录结构

```
hq-mksports-resource/
├── pages/          # 存放独立的 HTML 页面文件
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个完整的 HTML 页面。
- `assets/`：按需存放页面所需的共用或独立资源。

## 页面归档说明

- 所有 HTML 页面均为独立可浏览的静态文件。
- 页面之间无依赖关系，可单独部署或预览。
- 归档内容不包含任何动态后端逻辑，仅包含前端展示代码。
- 页面历史版本可通过 Git 提交记录追溯。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/hq-mksports-resource.git
   ```
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件即可查看效果。
3. 如需部署，可将整个仓库或 `pages/` 目录上传至任意静态服务器。
